Facebook Access Token

To get a facebook access token you need to register as a Facebook Developer here.

After that you can go to this page and request a access token by cliking "Get Token" and then "Get Access Token".Make sure "user_events" is filled and click "Get Access Token" button and then copy the access token inte to application. Note that the token often expire after a few hours and you need to request a new one.

Longer Access token

If you create an app on Facebook developer and use that to create an access token you can make the token last up to 3 months, however make sure to click "user_events" when it ask for permissions.
